Getting Started

Find your path into Runestone

Everyone is welcome here. Pick the description that fits you best and follow the steps — you'll be reading an interactive book in minutes.

Independent learner

You're teaching yourself and choosing your own book.

  1. 1

    Browse the Library

    Explore every open, interactive book we host.

  2. 2

    Note the book's code

    Each Library card shows a short course code — you'll need it to sign up.

  3. 3

    Register for that book

    Create a free account and enter the code.

  4. Start reading

    Your book opens right where you left off, every time.

    Open the Library

Student in a class

Your teacher gave you a course code to join.

  1. 1

    Register with your course code

    Sign up and enter the exact code your instructor gave you.

  2. Start reading

    You're in your class's book — assignments and all.

    Sign up & join

Instructor

You want to teach with a Runestone book of your own.

  1. 1

    Browse the Library

    Find a book that fits your course.

  2. 2

    Note the book's code

    You'll register for it before building your own course from it.

  3. 3

    Register for that book

    Create your free account. You become an instructor in the next step.

  4. Create your course

    Spin up your own copy, invite students, and track their progress.

    Create a course
